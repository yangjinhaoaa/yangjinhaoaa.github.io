[Linux]ssh连接的建立

win10通过ssh连接ubuntu

被连接机器配置

  • 待连接机器,与主机处于同一局域网
  • 检查ubuntu是否安装ssh 得到如下结果 说明已经安装ssh无需再次安装
  • 如果没有安装ssh用以下命令安装sudo apt-get install openssh-server
  • 开启ssh服务sudo service ssh start
  • ifconfig 获取机器IP地址

    连接机器配置

  • 进入cmd输入ssh -p port username@ip 默认端口22 username登录用户名 ip待连接机器ip地址
